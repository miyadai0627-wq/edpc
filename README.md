# edpc
正式名称：english-dictation-and-pronunciation-checker
# アプリ名（仮）：Custom Voice Check（カスタム・ボイス・チェック）

## 概要
自分だけの英語フレーズを登録し、リスニングと発音の練習ができるブラウザ完結型のWebアプリケーションです。
外部のデータベースや有料APIに依存せず、ブラウザの標準機能のみを活用して、単語帳の管理から音声合成（お手本再生）、音声認識（発音チェック）までを行うことができます。

## 主な機能

My単語帳機能 (CRUD操作): 練習したい英単語やフレーズと日本語訳を入力し、ブラウザ上に保存・一覧表示・削除ができます。

お手本リスニング (Text-to-Speech): 登録した英語フレーズを、ネイティブな発音で読み上げます。

発音チェッカー (Speech-to-Text): マイクを使って自分が発音した音声をテキスト化し、登録したフレーズと一致しているかを判定します。

## 使用技術

フロントエンド: HTML, CSS, JavaScript (Vanilla JS)

データ保存: LocalStorage

音声処理: Web Speech API (SpeechSynthesis / SpeechRecognition)

バージョン管理: Git / GitHub

## このプロジェクトの開発ポイント
フロントエンドの基礎的な技術を組み合わせ、実用的なインタラクティブ・アプリを構築しました。
データの追加・表示・削除といったタスク管理アプリの基本ロジックを土台にしつつ、そこにデバイスのマイクやスピーカーを制御するブラウザAPIを連携させることで、よりリッチなユーザー体験（UX）を実現しています。
