# edpc
正式名称：english-dictation-and-pronunciation-checker
# アプリ名（仮）：Custom Voice Check（カスタム・ボイス・チェック）

## 概要
自分だけの英語フレーズを登録し、リスニングと発音の練習ができるブラウザ完結型のWebアプリケーションです。
外部のデータベースや有料APIに依存せず、ブラウザの標準機能のみを活用して、単語帳の管理から音声合成（お手本再生）、音声認識（発音チェック）までを行うことができます。

## 💡 背景とコンセプト (Background & Concept)

### 開発のきっかけ：自身の英語学習における「リアルな課題」
本アプリは、開発者自身の英語学習における「インプットとアウトプットのギャップ」を埋めるために誕生しました。

文法や単語の基礎知識は既存の学習アプリでインプットできるものの、「実際にフレーズを口に出して定着させる」ための反復練習が圧倒的に不足していると感じていました。
流行りのAI英会話ツールも試しましたが、会話のテンポに流されてしまい、「特定のフレーズを自分が納得いくまで何度も反復練習する」「発音の正確さだけをシンプルかつ厳密に判定してもらう」というストイックな用途には不向きでした。

### コンセプト：徹底反復！自分専用フレーズ・スパーリングアプリ
そこで、「自分が本当に使いたいフレーズ」だけを登録し、AIの相槌や余計なUIに邪魔されることなく、ひたすら「お手本を聞く → 発音する → 判定する」を繰り返せるスピーキング特化型の反復練習ツールを開発しました。

**【ターゲット層】**
- 基礎学習は終えており、スピーキングの「瞬発力」を鍛えたい人
- 日々忙しく働く中で、通勤時間や寝る前などの限られたスキマ時間を使って、効率よくアウトプットのトレーニングをしたい人
- AI英会話での実践練習の前に、まずは決まったフレーズを自信を持って言えるようになりたい人

**【こだわったポイント】**
フロントエンド（HTML/CSS/JavaScript）の技術とブラウザ標準のWeb Speech APIを活用し、外部データベースや有料APIに依存しない構成にしています。これにより、アカウント登録や設定の手間なく、ブラウザを開いてすぐに直感的なワンタップ操作で練習を始められる軽量な体験（UX）を実現しました。


## 主な機能

My単語帳機能 (CRUD操作): 練習したい英単語やフレーズと日本語訳を入力し、ブラウザ上に保存・一覧表示・削除ができます。

お手本リスニング (Text-to-Speech): 登録した英語フレーズを、ネイティブな発音で読み上げます。

発音チェッカー (Speech-to-Text): マイクを使って自分が発音した音声をテキスト化し、登録したフレーズと一致しているかを判定します。

## 使用技術

フロントエンド: HTML, CSS, JavaScript (Vanilla JS)

データ保存: LocalStorage

音声処理: Web Speech API (SpeechSynthesis / SpeechRecognition)

バージョン管理: Git / GitHub

## このプロジェクトの開発ポイント
フロントエンドの基礎的な技術を組み合わせ、実用的なインタラクティブ・アプリを構築しました。
データの追加・表示・削除といったタスク管理アプリの基本ロジックを土台にしつつ、そこにデバイスのマイクやスピーカーを制御するブラウザAPIを連携させることで、よりリッチなユーザー体験（UX）を実現しています。
